# Decoder
## General info 
Decoder is a free and open-source messaging application that operates on the Matrix protocol. The app is designed to provide a user-friendly experience while prioritizing security and decentralization.

## Git
### Branch flow

```
<type>/<project-prefix><ticket-number>-<short-desc>
```

#### Types:

- task
- fix

#### Examples:

- `task/frontend5-add-signin-page`
- `task/backend12-add-filters`
- `fix/common16-fix-signup-validation`

### Commit flow

```
<project-prefix>-<ticket-number>: <modifier> <desc>
```

#### Modifiers:

- `+` (add)
- `*` (edit)
- `-` (remove)

#### Examples:

- `frontend-5: + form component`
- `backend-12: * filter markup`
- `shared-16: - require prop for nickname field`
